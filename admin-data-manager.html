Delete Temporarily — marks the record as deleted and hides it from users immediately. It stays in Firestore and can be fully restored anytime.
Erase Permanently — removes the record from Firestore forever with no recovery. For Users, all linked data (investments, transactions, deposits, withdrawals, referrals) is also erased.
Firebase Auth accounts cannot be deleted from the browser. When you permanently erase a user, their Firestore data is wiped and they are flagged as authDeletePending. Go to Firebase Console → Authentication to delete their login account, or set up a Cloud Function to handle it automatically.
